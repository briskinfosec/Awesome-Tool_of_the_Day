#  Category 1 - Web Application

* [RVuln](https://github.com/iinc0gnit0/RVuln/blob/master/installation.md) - RVuln is Multi-threaded Web Application Vulnerability Scanner written in Rust.

* [BlackDir-Framework](https://github.com/RedVirus0/BlackDir-Framework.git) - Web Application Vulnerability Scanner.

* [Spaghetti](https://github.com/infosecsecurity/Spaghetti.git) - It is an Open Source web application scanner, it is designed to find various default and insecure files, configurations, and misconfigurations. Spaghetti is built on python2.7 and can run on any platform which has a Python environment.

* [Wfuzz](https://github.com/gwen001/wfuzz.git) - WFUZZ is used to discover common vulnerabilities in web applications through the method of fuzzing.
